# 演示脚本与路演（全栈版）

## 30 秒（展台）

> PhotoMate：会**自己走过来**的摄影助理。  
> 你递手机，它帮你构图，**灵巧手点快门**，照片在你手机里。  
> 也能用自带相机、打印照片。扫码看方案。

动作：现场走一段导航 → 递手机 → 点快门（或播预录）。

---

## 3 分钟（评委）

1. **问题**（20s）——不会拍、没机位、隐私  
2. **方案**（40s）——接手机，不是固定拍照机；概念图  
3. **现场**（90s）  
   - Dashboard 地图与航点  
   - 导航到合影点  
   - 接手机 → 点快门 → 还手机  
   - （有则）Insta360 / 打印  
4. **技术**（40s）——Galbot 导航 + 双臂 + Agent 状态机 + `/docs`  
5. **商业**（30s）——活动租赁 To B  

---

## 完整互动流程（7/10）

```
嘉宾走近
 → 30 秒介绍 + 邀扫 /docs（项目介绍网页）
 → 选：「用自己手机」或「机器人相机」
 → 机器人导航到合影点
 → 手机模式：递手机 → 构图 → 灵巧手点快门 → 归还
 → 或相机模式：拍摄 → 打印/二维码
 → 合影留念、收反馈
```

---

## PPT 页（全栈）

1. 封面 + 概念图  
2. 痛点  
3. 双模式架构图（手机 / Insta360）  
4. 硬件：夹具 + 灵巧手 + 打印机  
5. 软件：Agent 状态机 + Dashboard  
6. **现场视频**（端到端，含点快门）  
7. 商业 + 二维码 `/docs`  

---

## 演示失败预案

用 7/9 录的 **完整链路** 视频兜底；口头说明现场环境与彩排一致。
