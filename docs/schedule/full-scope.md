## 总目标（7/9 力争完成，7/11 傍晚硬截止）

到路演前，PhotoMate 能完整演示：

```
巡航 / 导航到合影点
  → 接待用户（语音或网页）
  → 选模式：用户手机 / Insta360
  → 【手机模式】接手机 → 构图 → 灵巧手点快门 → 还手机
  → 【相机模式】取流 → 构图 → 拍照 → 可选打印 → 递照片 / 二维码
  → Dashboard 全程可见地图与状态
  → /docs 嘉宾随时扫码看方案
```
