# 名词说明

## 「嘉宾 docs」是什么？

就是你在浏览器里打开的这个页面：

**`http://<机器人或Jetson的IP>:8000/docs`**

- 顶栏 + 左栏目录 + 白底正文  
- 内容来自 `docs/plan/AI草案整理1.md`、`AI草案整理2.md` 自动渲染  
- 首页使用新流程前端 `webs/flow/assets/` 中的产品与场景素材  

给评委、观众、路人扫码看的 **项目介绍网站**，不是让你去读 `docs/` 文件夹里的 md。

和 **`/` 指挥台** 的区别：

| 路径 | 给谁 | 干什么 |
|------|------|--------|
| `/` | 你 / 操作员 | 看地图、位姿、点航点、控导航 |
| `/docs` | 嘉宾 / 评委 | 读方案、看概念图、理解故事 |

---

## 其它常用词

| 词 | 意思 |
|----|------|
| Dashboard / 指挥台 | 同上，`/` 网页 |
| 计划书 | `docs/plan/` 里两份会议纪要 md |
| Jetson | 机器人机载电脑，跑 PhotoMate 服务 |
| 航点 | `config/waypoints.yaml` 里的合影位置 |
| mock | `config/app.yaml` 里 `robot.mock: true`，无真机假数据演示 |
